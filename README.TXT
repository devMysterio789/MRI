■ はじめに

　このディスクは、医療画像を提供するディスクです。
　
　ディスク内には、医学的機密情報が含まれています。あなたが意図した受取人あるいは
　許可されたユーザでない場合は、このラベルに示された施設と連絡をお取りください。


■ 使用方法

　ディスク内にあるSTART.BATを実行すると、PDIメディアランチャーが起動します。

　【 医療画像の閲覧方法 】
　　　　　
　　１．Webブラウザで画像を簡易的に見る（標準）
　　　
　　　　・「簡易的に画像を見る(JPEG)」をクリックします。

　　２．PDIビューワーで画像を専門的に詳しく見る（多機能）
　
　　　　・「専門的に画像を見る(DICOM)」をクリックします。


　● Webブラウザ
　　Webブラウザは簡易的に患者データを参照することができます。
　　また、画像上で右クリック→パソコンに
　　取り込んだりすることが簡単にできます。

　● PDIビューワー
　　PDIビューワーはディスク内の患者データを表示するためのビューワーです。
　　使用方法についてはPDIビューワーのヘルプをご覧ください。

　【 その他のファイルの閲覧方法 】

　　ディスク内に医療画像以外の添付ファイルが格納されている場合、これらの
　　ファイルを閲覧することが出来ます。      

　　１．一覧から添付ファイルを確認する

　　　　・「添付ファイル一覧表」をクリックします。

　　２．添付ファイルのフォルダを直接開いて確認する
　　
　　　　・「添付ファイルを見る」をクリックします。


■ 使用注意

  当コンテンツではJavaScriptを使用しております。お使いのブラウザでJavaScript
　を無効にされている場合、正しく表示されないことがあります。ご覧になる際には
　ブラウザ設定でJavaScriptを有効にしてください。
　詳しくは、「Javaスクリプトの設定方法」をご覧ください。
　

  当コンテンツではスタイルシートを使用しております。お使いのブラウザでスタイ
　ルシートを無効にされている場合、正しく表示されないことがあります。ご覧にな
　る際にはブラウザ設定でスタイルシートを有効にしてください。
　詳しくは、「スタイルシートの使用について」をご覧ください。


  ※ Microsoft Edge(Legacy版)を使用している方へ

　当コンテンツはMicrosoft Edge(Legacy版)に対応しておりません。
　ご覧になる際にはMicrosoft Edge(Chromium版)にバージョンアップするか、
　Internet Explorerを使用してください。

■ こんなときは
　
　● 自動起動されなかったとき

　　Windows 7の場合

　　・「スタート」→「コンピュータ」をクリックします。
　　・「患者診療情報Disc」を右クリックし、「開く」を選びます。
　　・ディスク内にあるSTART.BATを実行し、PDIメディアランチャーを起動します。

    Windows 8、Windows 8.1、Windows 10の場合

　　・デスクトップ画面で[Windowsロゴ]キー＋[E]キーを押し、エクスプローラを表示します。
　　※フォルダツリーの「クイックアクセス」が選択されている場合は、「PC」をクリックします。
　　・「患者診療情報Disc」を右クリックし、「開く」を選びます。
　　・ディスク内にあるSTART.BATを実行し、PDIメディアランチャーを起動します。


　● OSのバージョンの確認方法

　　OSのバージョンがわからない場合は以下の方法で確認してください。

　　・[Windows]キーを押しながら、[R]キーを押します。
　　・「ファイル名を指定して実行」のポップアップが表示されたら、「winver」 と入力し 「OK」をクリックします。
　　・表示された「Windowsのバージョン情報」画面から確認してください。


　● ブラウザのセキュリティの設定

　　Internet Explorerの場合、Internet Explorerを起動し、以下の設定を行ってください。

　　・「ツール」メニューの「インターネットオプション」を選びます。
　　・「インターネットオプション」ウィンドウで「詳細設定」のタブをクリックします。 
　　・「設定」ウィンドウ内の「セキュリティー」グループの「マイコンピュータでの、
　　　CDのアクティブコンテンツの実行を許可する。」のチェックボックスをチェックして下さい。 

　　Edgeの場合、以下の設定を行ってください。

　　・デスクトップのスタートボタンの隣にある検索ボックス[ここに入力して検索]に、
　　　「インターネット オプション」と入力して検索結果に表示される「インターネット オプション」を選びます。 
　　・「インターネットオプション」ウィンドウで「詳細設定」のタブをクリックします。 
　　・「設定」ウィンドウ内の「セキュリティー」グループの「マイコンピュータでの、
　　　CDのアクティブコンテンツの実行を許可する。」のチェックボックスをチェックして下さい。


　● Javaスクリプトの設定方法

　　Internet Explorerの場合、Internet Explorerを起動し、以下の設定を行ってください。

　　・「ツール」メニューの「インターネットオプション」を選びます。 
　　・「インターネットオプション」ウィンドウで「セキュリティ」のタブをクリックします。 
　　・「Webコンテンツのゾーン」の「インターネット」が選択されているのを確認し､
　　　「レベルのカスタマイズ」をクリックします。 
　　・「セキュリティの設定」ウィンドウで「スクリプト」→「アクティブスクリプト」設定を
　　　「有効にする」にチェックします。

　　Edgeの場合、以下の設定を行ってください。

　　・デスクトップのスタートボタンの隣にある検索ボックス[ここに入力して検索]に、
　　　「インターネット オプション」と入力して検索結果に表示される「インターネット オプション」を選びます。
　　・「インターネットオプション」ウィンドウで「セキュリティ」のタブをクリックします。
　　・「Webコンテンツのゾーン」の「インターネット」が選択されているのを確認し､ 
　　　「レベルのカスタマイズ」をクリックします。
　　・「セキュリティの設定」ウィンドウで「スクリプト」→「アクティブスクリプト」設定を
　　　「有効にする」にチェックします。

　　　または、Microsoft Edgeを起動し、以下の設定を行ってください。
　　　・画面右上に「…」で表示されているメニューアイコンをクリックし、「設定」を選びます。
　　　・設定画面の左側に表示されている「Cookieとサイトのアクセス許可」をクリックします。
　　　・サイトのアクセス許可から「JavaScript」と書かれた個所をクリックします。
　　　・「JavaScript」の「許可」を有効にします。

　● スタイルシートの使用について

　　Internet Explorerの場合、Internet Explorerを起動し、以下の設定を行ってください。

　　・「ツール」メニューの「インターネットオプション」を選びます。 
　　・「全般」のタブを選択し右下の「ユーザー補助」ボタンをクリックします。 
　　・表示された画面ですべてのチェックをはずし、「OK」をクリックします。 

　　Edgeの場合、以下の設定を行ってください。

　　・デスクトップのスタートボタンの隣にある検索ボックス[ここに入力して検索]に、
　　　「インターネット オプション」と入力して検索結果に表示される「インターネット オプション」を選びます。
　　・「全般」のタブを選択し右下の「ユーザー補助」ボタンをクリックします。
　　・表示された画面ですべてのチェックをはずし、「OK」をクリックします。
